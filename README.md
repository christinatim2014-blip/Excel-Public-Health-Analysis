# Excel Public Health Trend Analysis

This project analyzes multi-year public health data from the Centers for Disease Control and Prevention (CDC) using Microsoft Excel.

The project focuses on:
- Cleaning and preparing raw data
- Analyzing trends by year
- Comparing results across U.S. states
- Building interactive PivotTables and charts

Tools used:
- Excel PivotTables
- Slicers
- Conditional formatting
- Dynamic charts

Skills Demonstrated:
- Data cleaning and preparation
- Filtering and selecting relevant records
- Trend analysis (year-over-year and multi-year trends)
- Comparative analysis across U.S. states
- Building PivotTables for data summarization
- Creating interactive dashboards using slicers
- Data visualization and insight communication
